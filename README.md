# AI Services Landing (Next.js + React)

Красивый лендинг под услуги (Google Apps Script, Telegram-боты, AI и платформы) с карточками, анимациями и модальным окном.

## Быстрый старт
```bash
npm i
npm run dev
```

Открой http://localhost:3000

## Что поменять под себя
- `lib/services.ts` — цены, описания, кейсы, контакты (Telegram/Email)
- `app/page.tsx` — заголовок/подзаголовок
- `app/globals.css` — цвета/шрифты/стили

## Деплой на Vercel
1) Залей в GitHub
2) Import в Vercel (Next.js auto)
3) Deploy — никаких env не нужно
